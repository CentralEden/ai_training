## 1. データサイエンティスト

### 🎯 役割・責任
- **データ分析**: ビジネス課題をデータで解決
- **仮説検証**: 統計的手法による意思決定支援
- **レポーティング**: 経営陣への分析結果報告
- **予測モデル構築**: 売上予測、需要予測等

### 💼 典型的な業務
```python
# 日常業務例
- 売上データの分析・可視化
- A/Bテストの設計・分析
- 顧客セグメンテーション
- 予測モデルの構築・評価
- ダッシュボード作成
```

### 🛠️ 必要スキル
**技術スキル**:
- Python/R による統計分析
- SQL によるデータ抽出
- 統計学・機械学習の基礎
- データ可視化 (Tableau, Power BI)
- A/Bテスト設計

**ビジネススキル**:
- 課題設定能力
- 仮説思考
- プレゼンテーション
- ドメイン知識

### 📈 キャリアパス例
```
入社1年目: ジュニアデータサイエンティスト
    ↓
3-5年目: シニアデータサイエンティスト
    ↓
5-8年目: リードデータサイエンティスト
    ↓
8年目以降: 
├── データサイエンス部門マネージャー
├── チーフデータオフィサー (CDO)
└── AIコンサルタント
```

### 💰 年収目安
- **ジュニア**: 400-600万円
- **シニア**: 600-1000万円
- **リード**: 800-1500万円
- **マネージャー**: 1000-2000万円

### 📚 推奨学習パス
1. **共通基礎コース** (5週間)
2. **専門分野**:
   - 時系列解析 (売上予測)
   - データエンジニアリング (データ基盤)
   - 画像認識 (商品分析)
3. **経験年数別**:
   - シニアDS向けコース (統計・機械学習深化)

## 🗺️ キャリアパス選択ガイド

### あなたに合ったキャリアパスは？

#### 📊 技術 vs ビジネス志向
```
技術深化志向 ←→ ビジネス応用志向
│                              │
├ リサーチサイエンティスト      ├ AIコンサルタント
├ MLエンジニア                 ├ AIプロダクトマネージャー  
├ データエンジニア             └ AI事業開発
└ 専門特化エンジニア
```

#### 🎯 志向性別おすすめキャリア

**研究・技術開発が好き**
- リサーチサイエンティスト
- 専門特化エンジニア (CV/NLP/音響/ロボティクス)

**システム構築・運用が好き**
- データエンジニア
- MLOpsエンジニア
- 機械学習エンジニア

**ビジネス課題解決が好き**
- データサイエンティスト
- AIコンサルタント
- AIプロダクトマネージャー

**新規事業・起業に興味**
- AI事業開発
- AIプロダクトマネージャー
- AIコンサルタント

### 🚀 キャリア変更・複合パス

#### よくあるキャリア変更パターン
```
データサイエンティスト → AIコンサルタント
機械学習エンジニア → MLOpsエンジニア
リサーチサイエンティスト → 専門特化エンジニア
AIプロダクトマネージャー → AI事業開発
```

#### 複合スキルの価値
- **技術 × ビジネス**: 高い市場価値
- **複数技術領域**: 希少性の高いスペシャリスト
- **マネジメント × 技術**: リーダーシップポジション
